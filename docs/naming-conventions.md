# Conventions de nommage Azure

Les noms définitifs devront inclure un suffixe personnel court afin de garantir leur unicité.

| Ressource | Modèle de nom |
|---|---|
| Resource Group | `rg-football-lakehouse-dev` |
| Storage Account | `stfootball<suffixe>` |
| Data Factory | `adf-football-<suffixe>-dev` |
| Databricks Workspace | `dbw-football-<suffixe>-dev` |
| Key Vault | `kv-football-<suffixe>` |

Région cible proposée : `France Central`, sous réserve de disponibilité des services et quotas de l'abonnement étudiant.

Environnement initial unique : `dev`.
